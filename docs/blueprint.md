# **App Name**: QR Info Reveal

## Core Features:

- QR Code Upload: Allow the user to upload an image containing a QR code.
- QR Code Reader: Read the QR code from the image. The app supports common QR code formats and encodings.
- Content Display: Display the information decoded from the QR code in a readable format. The display adjusts based on content type (text, URL, etc).
- Smart Titling: Use a tool to identify the type of content inside the QR code and generate a relevant title.

## Style Guidelines:

- Primary color: Soft Lavender (#D0B8F3) to give a gentle and trustworthy feel.
- Background color: Very light gray (#F5F5F5), almost white, for a clean and modern look.
- Accent color: A muted purple (#946CB2) that will be used for highlights and active states.
- Clean and readable sans-serif fonts.
- Simple and modern icons for actions like "upload" and "copy to clipboard".
- A clean and well-spaced layout to allow for readability and focus on the QR code content.
- Subtle animations on loading and when displaying content.
